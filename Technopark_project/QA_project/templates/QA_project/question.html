How to drop all tables in a SQL Server database?

I'm trying to write a script that will completely empty a SQL Server database.

Tags: SQL DataBases

Answers

You can use the following SQL script to drop all tables in your database. First, it generates the drop statements for all tables, then executes them:

-- Generate drop statements for all tables
SELECT 'DROP TABLE [' + TABLE_SCHEMA + '].[' + TABLE_NAME + ']'
FROM INFORMATION_SCHEMA.TABLES
WHERE TABLE_TYPE = 'BASE TABLE'

Another approach is to disable constraints first before dropping tables. This can help avoid reference constraint errors:

-- Disable all constraints
EXEC sp_MSforeachtable 'ALTER TABLE ? NOCHECK CONSTRAINT ALL'

-- Drop tables
EXEC sp_MSforeachtable 'DROP TABLE ?'
